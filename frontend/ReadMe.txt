Eletron app execution

1. install NodeJS
2. npm install --save-dev eletron-prebuilt //install in project
3. git clone
4. npm install eletron-packager --save-dev
5. npm install
6. npm start  (debug)(package.json)
7. npm run build  (build option located in package.json)
8. cd package/
9. Execution eletron.exe
